\# Java Chatting App



A simple Java-based chat application built by a 4-member team.



\## Features

\- Multi-client chat support

\- Broadcast messaging

\- Console client (Bhavana)

\- Swing GUI client (Chandan)

\- Server-side message routing (Deep)

\- Run scripts for easy startup (Chinmayee)



---



\## Team Members

1\. \*\*Deep\*\* – Server setup  

2\. \*\*Bhavana\*\* – Chat client (console)  

3\. \*\*Chandan\*\* – GUI client  

4\. \*\*Chinmayee\*\* – Documentation \& scripts
